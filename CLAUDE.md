# git-cheatsheet

Single-page Git & GitHub CLI command reference cheatsheet.

## Stack
- Single `index.html` — no build step, no dependencies
- GitHub Pages deployment from `main` branch root

## Site URL
https://marcvig.github.io/git-cheatsheet/

## Design
- Matches the nvim-cheatsheet / tmux-cheatsheet suite aesthetic
- Primary accent: #1D9E75 (teal-green)
- Font: JetBrains Mono (commands), Segoe UI (UI)
- Dark default, light mode toggle with localStorage persistence

## Features
- Tab switcher: All | Git | GitHub CLI
- Live search with yellow highlight on matches
- Command count updates dynamically
- Confetti easter egg (🌿 button)
- Back-to-top button
- Full SEO: meta, OG, Twitter card, JSON-LD (TechArticle), sitemap, robots.txt
